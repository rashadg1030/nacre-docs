Nacre is a structured, composable web framework for Haskell.

## Why Nacre?

- **Type-safe routes** - Define your API shape once, get type-checked handlers
- **Composable** - Build complex APIs from simple, reusable pieces
- **Layered like nacre** - Requests, responses, routes, actions, and middleware stack together beautifully

## Quick Example

```haskell
-- Define what comes in
input = request
  & method GET
  & path (lit "hello" *> capture @Text "name")

-- Define what goes out
output = response
  & status 200
  & body @Text

-- Create the route
route = input :-> output

-- Bind the handler to create an action
action = route := \Input{..} ->
  pure $ "Hello, " <> pathData <> "!"
```
