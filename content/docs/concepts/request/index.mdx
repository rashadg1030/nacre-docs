A request describes the shape of an incoming HTTP request. Each aspect of a request has its own DSL for type-safe composition.

```haskell
input
  = request
  & method GET
  & path do
      lit "users"
      userId <- capture @Int "id"
      pure userId
  & query do
      page <- page .= param @Int "page"
      pure page
  & headers none
  & body none
  & security none
```
