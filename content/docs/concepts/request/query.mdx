---
title: Query
---

The Query DSL extracts and validates query string parameters.

```haskell
query do
  page <- page .= param @Int "page"
  limit <- limit .= param @Int "limit"
  pure Pagination {..}
```

## Required Parameters

```haskell
param @Int "page"
param @Text "search"
```

## Optional Parameters

```haskell
optional @Int "page"
```

## Default Values

```haskell
withDefault 1 (param @Int "page")
```

## Building Records

Use `RecordWildCards` to build records from query parameters:

```haskell
data Pagination = Pagination
  { page :: Int
  , limit :: Int
  }

query do
  page <- page .= param @Int "page"
  limit <- limit .= param @Int "limit"
  pure Pagination {..}
```
