---
title: Body
---

The Body DSL specifies the expected request body type.

```haskell
body @Value
body @CreateUserRequest
body @Text
```

## JSON Body

Parse the request body as JSON into a type:

```haskell
data CreateUser = CreateUser
  { name :: Text
  , email :: Text
  }

body @CreateUser
```

## Raw Body

Accept raw text or bytes:

```haskell
body @Text
body @ByteString
```

## No Body

When no body is expected:

```haskell
body none
```
