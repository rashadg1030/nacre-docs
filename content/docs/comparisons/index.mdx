How Nacre compares to other web frameworks.

| Feature | Nacre | Servant | FastAPI |
|---------|-------|---------|---------|
| **Type Safety** | Full compile-time | Full compile-time | Runtime (Pydantic) |
| **Language** | Haskell | Haskell | Python |
| **API Style** | Value-level DSL | Type-level DSL | Decorators |
| **Learning Curve** | Moderate | Steep | Gentle |
| **Runtime Performance** | Excellent | Excellent | Good |
| **Compile Time** | Moderate | Slow | N/A |
| **Community** | New | Established | Large |
